Vocabulary

  • anachronism /əˈnækrənɪzəm/ noun

    something that exists or appears in the wrong historical period; a chronological inconsistency

    The AI model contained anachronisms, using modern terminology in texts supposedly from 1920.

    Also: chronological error, temporal inconsistency, misplacement

  • curate /ˈkjʊəreɪt/ verb

    to carefully select, organize, and present items; to act as a curator of a collection

    The team meticulously curated hundreds of billions of pre-1931 tokens from public domain sources.

    Also: select, organize, assemble, compile

  • slop /slɑːp/ noun

    low-quality, poorly made content; digital refuse or waste material

    Many argue the internet is polluted by AI slop that lacks originality.

    Also: junk, garbage, waste, dreck

  • baseline /ˈbeɪslaɪn/ noun

    a reference standard or starting point against which other things are measured or compared

    The new model performed better than the FineWeb baseline in our tests.

    Also: benchmark, reference point, standard, criterion

  • finetune /ˌfaɪnˈtjuːn/ verb

    to make small adjustments to a machine learning model to improve its performance on specific tasks

    Researchers can finetune base models to create specialized versions for different applications.

    Also: optimize, adjust, refine, calibrate

  • meticulously /məˈtɪkjələsli/ adverb

    in a careful, precise, and thorough manner; with great attention to detail

    The data was meticulously curated to ensure historical accuracy.

    Also: carefully, precisely, thoroughly, scrupulously

  • RLHF /ɑːr ɛl eɪtʃ ɛf/ noun

    Reinforcement Learning from Human Feedback; a technique for training AI models using human preferences and corrections

    RLHF was used to improve the model's responses based on historical text conventions.

    Also: human feedback alignment, preference learning

  • completions API /kəmˈpliːʃənz ˈeɪ pi ˈaɪ/ noun

    a programming interface that generates text completions based on a given prompt; often deprecated in favor of chat interfaces

    Some developers miss the completions API and want to experiment with base model variations.

    Also: text generation API, prompt completion endpoint

Slang & Jargon

low-background tokens

training data from historical periods with minimal modern contamination; clean, period-appropriate text without anachronisms

AI slop

low-quality, mass-produced AI-generated content that pollutes the internet with poor writing and originality

agent engineering

the practice of designing and building autonomous AI agents that can perform complex tasks independently

E2E harness

end-to-end testing framework that evaluates an entire system or pipeline from start to finish

automerge

automatic merging of model weights or code changes without manual intervention

alignment failure

when an AI model behaves in ways contrary to its training objectives or human values

base model

a foundational pre-trained language model before finetuning or specialization for specific tasks

mfw

my face when; internet slang expressing a reaction or emotion to a situation

SOTA

state-of-the-art; the highest level of development at a particular time

ChatML

a chat markup language format for structuring conversational AI training data and prompts
